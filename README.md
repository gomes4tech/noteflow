# 📚 NoteFlow

> Organizador de estudos pessoal — simples, bonito e direto ao ponto.

NoteFlow é uma aplicação web desenvolvida para ajudar estudantes a organizarem sua rotina de estudos. Com uma interface escura e moderna, reúne em um só lugar notas, metas, matérias e calendário semanal.

---

## ✨ Funcionalidades

- 🔐 **Login e perfil** — cadastro com nome, e-mail, cidade, bio e metas semanais
- 🏠 **Dashboard** — visão geral do progresso, dias marcados e metas em destaque
- 📝 **Notas** — criação e organização de anotações por matéria
- 📅 **Calendário semanal** — controle visual da semana de estudos
- 📖 **Matérias** — cadastro e gerenciamento das disciplinas
- 🎯 **Metas** — acompanhamento de objetivos com barra de progresso visual
- ⚙️ **Configurações** — edição de perfil e reset completo dos dados

---

## 🛠️ Tecnologias

- HTML5
- CSS3 (tema escuro, fonte customizada, design responsivo)
- JavaScript (vanilla)
- localStorage (persistência de dados no navegador)

```
NoteFlow/
```

---

## 🚀 Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/gomes4tech/NoteFlow.git
   ```
2. Abra o arquivo `login.html` no navegador
3. Crie sua conta e comece a organizar seus estudos!

> Não é necessário instalar nada. O projeto roda direto no navegador.
